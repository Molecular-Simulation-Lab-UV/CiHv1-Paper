DATA & FILE OVERVIEW
---------------------

File naming convention: Each file name consists of the figure number and the panel number that was generated using the data file. 
This is followed by the description of the data file.

DATA:
1. Folder name:	MODELS.
	Contents:It contain the active and resting models of the monomer CiHv1 generated by modeller program.
	format: pdb and psf files. 

2. Folder name: namd.
	Contents: It contain namd files for simulation, also toppars files.
	format: namd, str an d prm.

3. Folder name: P1.
	P1 folder contain the datasets of fig-1B, fig-1C, fig_2A, fig_2B, figS2A, figS2B. 
	Contents:It contain the dataset of the position of arginines, D160 and N4 in the differents CiHv1 simulation and states (3 simulations runs). Also contain the tcl script to 	calculated the positions (P1.tcl).
	format: dat, tcl.
	column 1: Z axis position (amstrong).
	column 2: P/P0 (probability).

4. Folder name:	PF.
	PF folder contain the datasets of fig-4C, figS6, figS7.
	Contents:It contain the dataset of the restrained and non-restrained Unitary osmotic permeability coefficient (pf) (3 dataset for each model). Also contain the Python script to 	calculated the Unitary osmotic permeability coefficient in bins (pf) and the input file to run the script (dn.py ; dnModule.py ; inputTest.in).
	format: npy, py, in.
	this is a npy format, a matrix that the diagonal column its de osmotic permeability coefficient in bins throught the pore (align in the Z axis).
	each bin its 2 amstrong of large align in the Z axis.

5. Folder name:	PME
	PME folder contain the dataset of fig-1, fig-1G, fig-2E, fig-2F, figS1, figS2E, figS2F.
	Contents:It contain the PMEpot dataset of the CiHv1 models, alsoo contain the tcl file to calculated the PMEpot and also the python file for add external potential and the python 	script to generate 1D potential along the Z axis (PMEpot.tcl ; 	externalPotential.py ; potential_1D.py)
	format: dx, tcl, py.
	The dx file was originally in KbT/e units and has been converted to volts (V). The external potential was first added to the data, and then the total values were transformed into 	voltage. 

6. Folder name: SALTBRIDGE
	SALTBRIDGE folder contain the dataset for fig-1D, fig-1E, fig-2C, fig-2D, figS2C, figS2D.
	Contents:It contain the dataset of distance between R3 and D160 and others interacctions (3 dataset for each model). Also contain the tcl script (SALTBRIDGE.tcl) to calculated 	the distances.
	format: dat, tcl.
	column 1: frame
	column (x): the next columns are the distances of the each interaction (Amstrong).

7. Folder name:	WATER_DENSITY
	WATER_DENSITY folder contain the dataset of fig-4A, figS6, figS7.
	Contents: WATER_DENSITY folder contain the dataset of restrained and non-restrained water density CiHv1 models (3 dataset for each model) and the tcl script to calculated the water 	density (DensP1.tcl).
	format: dat, tcl.
	column 1: Z axis position (amstrong).
	column 2: The number of water molecules per bin over time.
